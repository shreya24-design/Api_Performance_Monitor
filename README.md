# API Performance Monitor

## Project Description
This project monitors website/API performance by measuring response times and status codes.

## Technologies Used
- Python
- Requests
- Pandas
- Matplotlib
- Google Colab

## Features
- Monitors multiple websites
- Measures response times
- Tracks status codes
- Generates CSV reports
- Creates performance graphs
